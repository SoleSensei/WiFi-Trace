# Wifi Tracing
Visualization of WiFi coverage with ray tracing algorithm

## Build and Launch

        make
        cd build/bin
        ./visualize [-c] [config_file]
        
        ./visualize --help

### Launch script 

        bash test.sh # build and run with config_file
